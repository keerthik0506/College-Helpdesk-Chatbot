# College-Helpdesk-Chatbot
Built backend logic in Python to handle and process student queries efficiently. Designed a structured database schema and developed REST APIs for seamless request-response communication. Focused on modular architecture, clean code practices, and reliable error handling.
Developed a backend-driven chatbot system to automate responses for common college-related queries such as admissions, fees, schedules, and events. Designed and implemented the core backend logic using Python, ensuring efficient request handling and structured data processing.

Architected a normalized database schema to store user queries, responses, and interaction logs for scalability and future analytics. Built RESTful APIs to manage request-response flow between frontend and backend systems, enabling smooth communication and real-time query resolution.

Focused on writing clean, modular, and maintainable code following proper separation of concerns. Implemented input validation, error handling, and logging mechanisms to ensure reliability and easy debugging.
